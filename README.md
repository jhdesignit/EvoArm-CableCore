## Overview

EvoArm CableCore is a fully mechanical, cable-driven assistive arm.

It operates using natural arm movement. No electronics. No software. No power required.

This version is the foundation of the EvoArm system.

---

## How It Works

* Arm straight → hand opens
* Arm bends → hand closes
* Cable tension controls finger movement
* Elastic returns fingers to open position

---

## Key Features

* Fully mechanical design

* No batteries or electronics
* 3D printable components

## Build Instructions

1. Download STL files from /stl
2. 3D print all components
3. Follow the assembly guide in /build-guide
4. Install cables through routing channels
5. Add elastic return bands
6. Adjust cable tension for grip performance

---

## Materials Required

* PLA or PETG filament
* Nylon cord or strong fishing line
* Elastic bands
* Screws, pins, and fasteners
